# Bar Chart Examples

This folder contains examples of bar chart visualizations, including:

- **barplots.ipynb** - Basic bar chart examples with custom styling
- **grouped_barplots.ipynb** - Grouped/stacked bar chart examples
- **horizontal-barplot-with-labels-the-economist.ipynb** - Horizontal bar charts in The Economist style

## Usage

Import shared utilities from the `src` folder:

```python
from src.styles.colors import GRAY1, BLUE1, RED1
from src.styles.swd import apply_swd_style
from src.utils.annotations import rainbow_text
```
