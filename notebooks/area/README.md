# Area Chart Examples

This folder contains examples of area chart visualizations:

- **area-chart-with-different-colors-for-positive-and-negative-values.ipynb** - Area charts with color coding
- **stacked-area-with-inflexion-arrows.ipynb** - Stacked area charts with annotations
- **lineplots-and-area-chart-the-economist.ipynb** - Area charts in The Economist style

## Usage

Import shared utilities from the `src` folder:

```python
from src.styles.colors import *
from src.styles.economist import apply_economist_style
```
