# Specialized Chart Examples

This folder contains examples of specialized and less common chart types:

- **dumbell-chart.ipynb** - Dumbbell/connected dot plots
- **ridgeline-graph-seaborn.ipynb** - Ridgeline plots using seaborn
- **slopegraph.ipynb** - Slopegraph visualizations
- **lollipop.ipynb** - Lollipop chart examples
- **swarmplot.ipynb** - Swarm plot examples

## Usage

Import shared utilities from the `src` folder:

```python
from src.styles.colors import *
```
