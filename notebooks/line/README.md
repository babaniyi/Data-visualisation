# Line Chart Examples

This folder contains examples of line chart visualizations:

- **line-chart-with-labels-at-line-end.ipynb** - Line charts with end-point labels
- **stacked-line-chart-with-labels.ipynb** - Stacked line chart examples

## Usage

Import shared utilities from the `src` folder:

```python
from src.styles.colors import *
from src.styles.swd import apply_swd_style
```
